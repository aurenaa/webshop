# WebShop Project 2025
Modern web application for online shopping built with Java technologies and RESTful services.

# Project Overview
WebShop is a full-stack e-commerce application developed as part of the Web Programming course at the Faculty of Technical Sciences. The application provides a complete online shopping experience with user management, product catalog, shopping cart, and order processing.

# Architecture
Backend: Java Servlets, RESTful Web Services (Jersey)
Frontend: HTML, CSS, JavaScript
Server: Apache Tomcat 9.0.73
Build Tool: Maven

# Features
User Management
User registration and authentication
Role-based access control (Admin/User)
Session management
User profile management
Product Catalog
Product listing with categories
Product search and filtering
Product details and images
Inventory management
Shopping Cart
Add/remove products from cart
Quantity management
Order Processing
Order creation and management
Order history
Order status tracking

Admin Panel
Product management (CRUD operations)
User management
